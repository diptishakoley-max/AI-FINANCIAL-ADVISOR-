## 🚀 Getting Started

### Prerequisites

- Python 3.10 or higher
- A [Google Gemini API Key](https://ai.google.dev/)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/diptishakoley-max/AI-FINANCIAL-ADVISOR-.git
   cd AI-FINANCIAL-ADVISOR-
   ```

2. **Create and activate a virtual environment**
   ```bash
   python -m venv venv

   # Windows
   venv\Scripts\activate

   # macOS / Linux
   source venv/bin/activate
   ```

3. **Install dependencies**
   ```bash
   pip install -r requirements.txt
   ```

4. **Set up your API key** — create a `.env` file in the project root:
   ```env
   GEMINI_API_KEY=your_google_gemini_api_key_here
   ```

5. **Run the application**
   ```bash
   streamlit run app.py
   ```

   The app will open at **http://localhost:8501**

## 📖 Usage

1. **Fill in your financial profile** in the sidebar:
   - Select profile type (Student / Professional / Other)
   - Enter income, expenses, existing savings, debts
   - Set financial goals and risk tolerance

2. **Click "Financial Analysis & Advice"** to generate:
   - Key financial metrics (savings, investment capacity, net worth)
   - Savings, debt-to-income, and expense ratio cards
   - Health indicators with visual progress bars
   - AI-generated personalized financial advice

3. **Explore the tabs:**
   - 📈 **Financial Advice** — AI-generated recommendations in categorized sections
   - 📊 **Visualizations** — Pie and bar charts with detailed breakdowns
   - 🎯 **Goal Planner** — Enter custom instructions for advanced goal-oriented plans
   - 💬 **Financial Chatbot** — Ask any finance-related question

4. **About Tool** section at the bottom summarizes purpose, use cases, and technologies.

## 🔍 Modules Overview

### `app.py`
Main Streamlit application handling page configuration, sidebar inputs, session state management, layout rendering (metrics, cards, tabs, chatbot), and the About section.

### `ai_advisor.py`
Three core AI functions powered by Google Gemini:
- `generate_financial_advice(user_data, analysis_data)` — comprehensive financial recommendations
- `generate_goal_plan(user_data, analysis_data, user_instructions)` — custom goal-oriented plans
- `finance_chatbot_response(user_data, analysis_data, user_query)` — interactive Q&A responses

### `finance_analysis.py`
Pure Python financial engine:
- `analyze_finances(user_data)` — returns savings, ratios, investment capacity, emergency fund targets, net worth, recommended allocations, and debt alerts

### `visualization.py`
Chart generation with Matplotlib and Seaborn:
- `plot_advised_financial_overview(user_data, analysis_data)` — dual-subplot figure with pie chart and bar chart

### `utils.py`
AI response parsing utilities:
- `split_advice_sections(advice_text)` — splits advice into titled HTML sections
- `split_goal_sections(goal_text)` — splits goal plans into titled HTML sections

### `styles.css`
Custom CSS featuring gradient backgrounds, animated card hover effects (translateY + shimmer), gradient text headers, styled sidebar, progress bars, chat bubbles, and responsive layouts.
